# bilingualism_semantics
This repo contains analysis code for Chen et al., 2026 "Bilingual Language Processing Relies on Shared Semantic Representations that are Modulated by Each Language"

The data used by these scripts, and reproduced analysis are available on our [CodeOcean Code Capsule](https://codeocean.com/capsule/9997376/tree/v1)

Preprocessed data and stimuli are available on [GIN](https://gin.g-node.org/denizenslab/narratives_bilingualism_zh_en_fMRI)
